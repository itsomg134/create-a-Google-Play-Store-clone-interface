# Play Store Clone

A modern, responsive Google Play Store clone built with React and Tailwind CSS. This project replicates the core UI/UX of the Google Play Store, featuring app browsing, categories, and search functionality.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Search Functionality** - Search bar for finding apps and games
- **Tab Navigation** - Browse different sections (For You, Games, Apps)
- **Featured Apps Carousel** - Showcases recommended applications
- **Category Browser** - Quick access to top app categories
- **Top Charts** - Displays trending and popular apps with rankings
- **Bottom Navigation** - Mobile-friendly navigation bar
- **Interactive UI** - Hover effects and smooth transitions throughout

## Tech Stack

- **React** - Frontend framework for building the UI
- **Tailwind CSS** - Utility-first CSS framework for styling
- **Lucide React** - Icon library for consistent, beautiful icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/play-store-clone.git
cd play-store-clone
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

The application provides several interactive sections:

- Use the **search bar** to find specific apps
- Click on **navigation tabs** to switch between different store sections
- Browse **featured apps** in the recommendation carousel
- Explore **categories** by clicking on category cards
- View **top charts** to see trending applications
- Use **bottom navigation** for quick access to different store sections

## Customization

You can easily customize the app data by modifying the following arrays in `PlayStoreClone.jsx`:

- `featuredApps` - Change featured applications
- `categories` - Modify app categories
- `topApps` - Update top chart rankings
